SPP Framework Tutorial

Welcome to the official tutorial for the Satya Portal Pack (SPP) Framework. This guide is designed to take you from a beginner to a pro in building modular, event-driven web applications using SPP.

Tip: If you are new to the framework, start with the Introduction to understand the architectural flow.

  5. Routing and Views: Defining Routes in `pages.yml`. The `ViewPage` Rendering Engine. SPA "Drop and Play".
  6. Forms & Validation: YAML/XML Form Definitions. Automatic Form Augmentation. Client and Server-side Validation.
  7. The Event System: Understanding Hooks. Registering and Firing Events. Extending Core Functionality.
